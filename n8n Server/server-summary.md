## Можно ли что-то ещё ставить?

**Да.**  
~80 ГБ диск и ~10 ГБ свободной RAM позволяют добавить, например:

- ещё Docker-контейнеры (мессенджеры, боты, свои приложения);
- системные пакеты: `sudo apt update && sudo apt install <пакет>`;
- дополнительные модели Ollama (место на диске нужно смотреть отдельно).

Рекомендация: тяжёлые вещи (большие БД, много моделей) сначала проверять по месту: `df -h` и `docker system df`.
